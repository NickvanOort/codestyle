TLDR¶

  • Don’t use os.path, use pathlib.Path instead, see paths

  • Don’t use print, use loguru.logger, see loguru

  • Use pydantic for dataclasses and settings management, see pydantic

  • Use Typehinting, and install black, flake8, isort, mypy, pep8-naming, flake8-annotations to format/lint your code with make, see Linting

  • Create your own types with either Protocol (see Typehinting) or pydantic (see Classes) if things get too complex

  • Manage your environment with poetry instead of conda, see Poetry

  • Use the data science cookiecutter as a starting point for project, see Data Science cookiecutter
